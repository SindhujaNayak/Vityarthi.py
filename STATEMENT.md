# Project Statement

**Project Name:** Python CLI To-Do List

**Date:** Sunday, November 23, 2025

## Objective

Create a simple command-line To-Do List application in Python. The program should let users add tasks, view all tasks, mark them as complete and remove tasks using an interactive menu.

## Problem Statement

Managing daily tasks is essential for productivity but can be challenging without a simple tool. This project addresses the need for a basic,easy-to-use and platform independent to do list manager for personal use especially suitable for beginners learning Python.

## Scope

- Interactive CLI (command-line interface)
- Basic task operations (add, view, mark as done, delete)
- No external package dependencies
- Self-contained Python script

## Expected Outcome

- A working Python script that enables users to effectively manage a simple set of tasks directly from the terminal.
- Clean and readable code suitable for beginners seeking to understand basic Python logic and data structures.
