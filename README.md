# Python CLI To-Do List

A simple command-line to-do list application in Python to help you manage tasks interactively through a text-based menu.

## Features

- Add new tasks
- View all tasks with their status (Done / Not Done)
- Mark tasks as done
- Delete tasks from the list
- Exit the program cleanly

## Requirements

- Python 3.x installed on your system

## Usage

1. Download or copy the script to your computer.
2. Open a terminal or command prompt.

5. Use the numbered menu prompts to add, view, mark as done, delete tasks, or exit.
